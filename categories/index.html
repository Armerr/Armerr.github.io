# 艺术哲学

2023年2月14日23:45:55

说起艺术哲学，对于人们可能更熟悉的是“美学”这个词，这确实是实在的，人们早已习惯于将“艺术”与“美”相提并论或者再根据传统美学的立场，将“艺术”置于“美”之下，这一点足以表明人心中的固执。美学原理何以作为第一原理横亘在艺术之上？美何以比艺术更为根本？一开始我就是要指出这个属于形而上学的错误，做出对传统美学的批判。

#
